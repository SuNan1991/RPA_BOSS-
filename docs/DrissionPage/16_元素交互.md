## 点击元素

### click() 和 click.left()

这两个方法作用是一样的，用于左键点击元素。可选择模拟点击或 js 点击。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `by_js` | `bool` | `False` | 指定点击行为方式 |
| `timeout` | `float` | `1.5` | 模拟点击的超时时间（秒） |
| `wait_stop` | `bool` | `True` | 点击前是否等待元素停止运动 |

**返回值**:
- `False` - 元素不可用、不可见时
- `True` - 其余情况

**示例**:

```python
# 对ele元素进行模拟点击
ele.click()

# 用js方式点击ele元素，无视遮罩层
ele.click(by_js=True)

# 如元素不被遮挡，用模拟点击，否则用js点击
ele.click(by_js=None)
```

**点击方式说明**:
- `by_js` 为 `None`: 优先用模拟方式点击，如遇遮挡、元素不可用、不可见、无法自动进入视口，等待直到超时后自动改用 js 方式点击
- `by_js` 为 `False`: 强制使用模拟点击，即使被遮挡也会点击元素位置
- `by_js` 为 `True`: 可无视任何遮挡，只要元素在 DOM 内，就能点击得到

### click.right()

此方法实现右键单击元素。

---

### click.middle()

此方法实现中键单击元素。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `get_tab` | `bool` | `True` | 是否返回新出现的 Tab 对象 |

**返回**: 新 Tab 对象或 `None`

**示例**:

```python
tab = ele.click.middle()
print(tab.title)
```

---

### click.multi()

此方法实现左键多次点击元素。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `times` | `int` | `2` | 点击次数 |

---

### click.at()

此方法用于带偏移量点击元素，偏移量相对于元素左上角坐标。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `offset_x` | `float` | `None` | x 轴偏移量 |
| `offset_y` | `float` | `None` | y 轴偏移量 |
| `button` | `str` | `'left'` | 要点击的键 |
| `count` | `int` | `1` | 点击次数 |

**示例**:

```python
# 点击元素右上方 50*50 的位置
ele.click.at(50, -50)

# 点击元素上中部
ele.click.at(offset_x=50)

# 和click()一致，但没有重试功能
ele.click.at()
```

### click.to_upload()

此方法用于点击元素，触发文件选择框并把指定的文件路径添加到网页。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `file_paths` | `str` `Path` `list` | 必填 | 文件路径 |
| `by_js` | `bool` | `False` | 是否用 js 方式点击 |

---

### click.to_download()

此方法用于点击元素触发下载，并返回下载任务对象。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `save_path` | `str` `Path` | 必填 | 保存路径 |
| `rename` | `str` | `None` | 重命名文件名 |
| `suffix` | `str` | `'left'` | 指定文件后缀 |
| `new_tab` | `bool` | `1` | 该下载是否在新 tab 中触发 |
| `by_js` | `bool` | `False` | 是否用 js 方式点击 |
| `timeout` | `float` | `None` | 超时时间（秒） |

**返回**: `DownloadMission` 下载任务对象

---

### click.for_new_tab()

在预期点击后会出现新 tab 的时候，可用此方法点击，会等待并返回新 tab 对象。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `by_js` | `bool` | `False` | 是否用 js 方式点击 |

**返回**: 新 Tab 对象

## 输入内容

### clear()

此方法用于清空元素文本，可选择模拟按键或 js 方式。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `by_js` | `bool` | `False` | 是否用 js 方式清空 |

---

### input()

此方法用于向元素输入文本或组合键。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `vals` | `Any` | `False` | 文本值或按键组合 |
| `clear` | `bool` | `False` | 输入前是否清空文本框 |
| `by_js` | `bool` | `False` | 是否用 js 方式输入 |

**示例**:

```python
# 输入文本
ele.input('Hello world!')

# 输入文本并回车
ele.input('Hello world!\n')
```

---

### 输入组合键

使用组合键或要传入特殊按键前，先要导入按键类 `Keys`。

```python
from DrissionPage.common import Keys

# 组合键
ele.input((Keys.CTRL, 'a', Keys.DEL))  # ctrl+a+del

# 常用组合键
ele.input(Keys.CTRL_A)  # 全选
ele.input(Keys.CTRL_C)  # 复制
ele.input(Keys.CTRL_V)  # 粘贴
```

## 拖拽和悬停

### drag()

此方法用于拖拽元素到相对于当前的一个新位置。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `offset_x` | `int` | `0` | x 轴偏移量 |
| `offset_y` | `int` | `0` | y 轴偏移量 |
| `duration` | `float` | `0.5` | 用时（秒） |

**示例**:

```python
# 拖动当前元素到距离50*50的位置，用时1秒
ele.drag(50, 50, 1)
```

---

### drag_to()

此方法用于拖拽元素到另一个元素上或一个坐标上。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `ele_or_loc` | `ChromiumElement` `Tuple` | 必填 | 另一个元素对象或坐标元组 |
| `duration` | `float` | `0.5` | 用时（秒） |

**示例**:

```python
# 把 ele1 拖拽到 ele2 上
ele1.drag_to(ele2)

# 把 ele1 拖拽到网页 50, 50 的位置
ele1.drag_to((50, 50))
```

---

### hover()

此方法用于模拟鼠标悬停在元素上。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `offset_x` | `int` | `None` | x 轴偏移量 |
| `offset_y` | `int` | `None` | y 轴偏移量 |

**示例**:

```python
# 悬停在元素右上方 50*50 的位置
ele.hover(50, -50)

# 悬停在元素中点
ele.hover()
```

## 执行 js 脚本

### run_js()

此方法用于对元素执行 js 代码，代码中用 `this` 表示元素自己。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `script` | `str` | 必填 | js 脚本文本 |
| `*args` | - | 无 | 传入的参数 |
| `as_expr` | `bool` | `False` | 是否作为表达式运行 |
| `timetout` | `float` | `None` | js 超时时间（秒） |

**返回**: 脚本执行结果

**示例**:

```python
# 用执行 js 的方式点击元素
ele.run_js('this.click();')

# 用 js 获取元素高度
height = ele.run_js('return this.offsetHeight;')
```

---

此方法用于添加初始化脚本，在页面加载任何脚本前执行。

### remove_init_js()

## 元素滚动

元素滚动功能藏在 `scroll` 属性中。

### scroll() 或 scroll.down()

使元素向下滚动若干像素。

```python
ele.scroll(30)
ele.scroll.down(30)
```

---

### scroll.up()

使元素向上滚动若干像素。

---

### scroll.right() / scroll.left()

向右或向左滚动。

---

### scroll.to_top() / scroll.to_bottom() / scroll.to_half()

滚动到顶部、底部或中间。

---

### scroll.to_rightmost() / scroll.to_leftmost()

滚动到最右边或最左边。

---

### scroll.to_location()

滚动到指定位置。

```python
ele.scroll.to_location(100, 300)
```

---

### scroll.to_see()

滚动页面直到元素可见。

---

### scroll.to_center()

尽量把元素滚动到视口正中。

## 列表选择

`<select>` 下拉列表元素功能在 `select` 属性中。

### select() 和 select.by_text()

按文本选择列表项。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `text` | `str` `list` | 必填 | 作为选择条件的文本 |
| `timeout` | `float` | `None` | 超时时间（秒） |

---

### select.by_value()

按 `value` 属性选择列表项。

---

### select.by_index()

按序号选择列表项，从 `1` 开始。

---

### select.cancel_by_text() / cancel_by_value() / cancel_by_index()

取消选择列表项。

---

### select.all() / select.clear() / select.invert()

全选、取消全选、反选（多选列表有效）。

---

### select.options

### select.selected_option / selected_options
