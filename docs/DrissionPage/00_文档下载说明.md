## 🚀 快速参考

### 基础使用流程

```python
from DrissionPage import ChromiumPage

# 1. 创建页面对象
page = ChromiumPage()

# 2. 访问网页
page.get('https://www.example.com')

# 3. 查找元素
ele = page.ele('#selector')

# 4. 元素操作
ele.click()
ele.input('text')
```

### 三种页面对象选择

| 对象 | 适用场景 | 速度 | 功能 |
|------|---------|------|------|
| **ChromiumPage** | 需要执行JavaScript、复杂交互 | 慢 | 强大 |
| **SessionPage** | 纯数据采集、API调用 | 快 | 有限 |
| **WebPage** | 需要两种模式结合 | 可切换 | 完整 |

### WebPage 模式切换

```python
from DrissionPage import WebPage

page = WebPage()

# d 模式处理登录
page.get('https://example.com/login')
page.ele('#username').input('user')
page.ele('#password').input('pass')
page.ele('#login-btn').click()

# 切换到 s 模式采集数据
page.change_mode(mode='s')
page.get('https://example.com/data')
items = page.eles('.item')
```
