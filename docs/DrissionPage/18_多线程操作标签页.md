# 多线程操作标签页

来源: https://drissionpage.cn/dp40docs/demos/actual/multithread

本示例演示如何使用多线程同时操作多个标签页进行数据采集。

---

## 完整代码

```python
from threading import Thread
from DrissionPage import ChromiumPage
from DataRecorder import Recorder

def collect(tab, recorder, title):
    """用于采集的方法

    :param tab: ChromiumTab 对象
    :param recorder: Recorder 记录器对象
    :param title: 类别标题
    :return: None
    """
    num = 1  # 当前采集页数

    while True:
        # 遍历所有标题元素
        for i in tab.eles('.title project-namespace-path'):
            # 获取某页所有库名称，记录到记录器
            recorder.add_data((title, i.text, num))

        # 如果有下一页，点击翻页
        btn = tab('@rel=next', timeout=2)
        if btn:
            btn.click(by_js=True)
            tab.wait.load_start()
            num += 1
        # 否则，采集完毕
        else:
            break

def main():
    # 新建页面对象
    page = ChromiumPage()

    # 第一个标签页访问网址
    page.get('https://gitee.com/explore/ai')
    # 获取第一个标签页对象
    tab1 = page.get_tab()

    # 新建一个标签页并访问另一个网址
    tab2 = page.new_tab('https://gitee.com/explore/machine-learning')
    # 获取第二个标签页对象
    tab2 = page.get_tab(tab2)

    # 新建记录器对象
    recorder = Recorder('data.csv')

    # 多线程同时处理多个页面
    Thread(target=collect, args=(tab1, recorder, 'ai')).start()
    Thread(target=collect, args=(tab2, recorder, '机器学习')).start()

if __name__ == '__main__':
    main()
```

## 代码解析

### 1. 采集函数 collect()

接收三个参数：
- `tab`: ChromiumTab 对象，用于控制标签页
- `recorder`: Recorder 记录器对象，用于保存数据
- `title`: 类别标题，用于区分不同来源的数据

**逻辑**:
1. 遍历当前页面的所有项目名称
2. 将项目名称和类别记录到 recorder
3. 查找"下一页"按钮
4. 如果有，点击翻页并继续
5. 如果没有，结束采集

---

### 2. 主函数 main()

**流程**:
1. 创建 ChromiumPage 对象
2. 第一个标签页访问 AI 类别页面
3. 新建第二个标签页访问机器学习类别页面
4. 创建 Recorder 记录器对象
5. 使用 Thread 开启两个线程同时采集不同类别的数据

## 关键要点

### Tab 对象的获取

```python
# 获取当前标签页
tab1 = page.get_tab()

# 新建标签页并获取
tab2 = page.new_tab('https://...')
tab2 = page.get_tab(tab2)
```

### 多线程的使用

```python
from threading import Thread

# 创建线程
Thread(target=函数名, args=(参数1, 参数2, ...)).start()

# 多个线程可以同时操作不同的标签页
Thread(target=collect, args=(tab1, recorder, 'ai')).start()
Thread(target=collect, args=(tab2, recorder, '机器学习')).start()
```

### 翻页操作

```python
# 查找下一页按钮
btn = tab('@rel=next', timeout=2)

if btn:
    # 点击按钮
    btn.click(by_js=True)
    # 等待新页面开始加载
    tab.wait.load_start()
```

## DataRecorder

`DataRecorder` 是一个用于数据记录的库，可以方便地将采集的数据保存到 CSV 或 Excel 文件。

**安装**:
```bash
pip install DataRecorder
```

**基本用法**:
```python
from DataRecorder import Recorder

# 创建记录器
recorder = Recorder('data.csv')

# 添加数据
recorder.add_data(('列1', '列2', '列3'))
```

---

## 应用场景

这种多线程采集模式适用于：
- 需要同时采集多个分类的数据
- 不同分类之间互不干扰
- 提高采集效率
