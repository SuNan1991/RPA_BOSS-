# 下载功能概述

来源: https://drissionpage.cn/dp40docs/download/intro

DrissionPage 提供了强大的文件下载管理功能。

能够主动发起下载任务，也能够对浏览器触发的下载任务进行管理。

---

## download() 方法

该方法可以主动发起下载任务，提供以下功能：
- 任务管理
- 多线程
- 大文件分块
- 自动重连
- 文件名冲突处理

页面对象、标签页对象、`Session` 对象都支持此方法。

---

## 基本用法

### SessionPage 下载

```python
from DrissionPage import SessionPage

page = SessionPage()
url = 'https://example.com/file.zip'
page.download(url, goal_path='downloads', rename='myfile.zip')
```

## 浏览器触发下载

### 使用 click.to_download()

```python
from DrissionPage import ChromiumPage

page = ChromiumPage()
page.get('https://example.com')

# 找到下载按钮
btn = page('下载按钮')

# 点击并触发下载
mission = btn.click.to_download(
    save_path='downloads',
    rename='myfile.zip'
)

# 等待下载完成
mission.wait()

# 打印下载路径
print(mission.path)
```

---

### 手动触发下载

```python
from DrissionPage import ChromiumPage

page = ChromiumPage()

# 设置下载路径
page.set.download_path('downloads')

# 点击触发下载
btn = page('下载按钮')
btn.click()

# 等待下载开始
mission = page.wait.download_begin()

# 等待下载完成
mission.wait()

print(f'下载完成: {mission.path}')
```

## 下载任务对象 DownloadMission

`DownloadMission` 对象用于管理下载任务。

### 常用属性

- `path` - 下载文件保存路径
- `url` - 下载地址
- `total_size` - 文件总大小
- `downloaded_size` - 已下载大小
- `status` - 下载状态

### 常用方法

- `wait()` - 等待下载完成
- `cancel()` - 取消下载
- `pause()` - 暂停下载
- `resume()` - 恢复下载

---

## 高级功能

### 断点续传

```python
mission = page.download(url, goal_path='downloads')

# 下载可以中断后继续
mission.pause()
mission.resume()
```

---

### 多线程下载

```python
# 自动使用多线程下载大文件
mission = page.download(url, goal_path='downloads', threads=5)
```

---

### 下载进度监听

```python
mission = page.download(url, goal_path='downloads')

while not mission.is_done:
    print(f'下载进度: {mission.downloaded_size}/{mission.total_size}')
    time.sleep(1)
```

---

## 文件名处理

### 自动重命名

```python
mission = btn.click.to_download(
    save_path='downloads',
    rename='new_name.zip'  # 重命名文件
)
```

### 处理文件名冲突

DrissionPage 会自动处理文件名冲突，在文件名后添加序号。
