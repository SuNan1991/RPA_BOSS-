## 📝 核心概念速查

### 三种页面对象

| 对象 | 用途 | 速度 | 功能 |
|------|------|------|------|
| **ChromiumPage** | 浏览器控制 | 慢 | 强大 |
| **SessionPage** | 数据包收发 | 快 | 有限 |
| **WebPage** | 双模式整合 | 可切换 | 完整 |

### 两种工作模式

- **d 模式 (Driver)**: 控制浏览器，功能强大但速度慢
- **s 模式 (Session)**: 收发数据包，速度快但功能有限

### 基本操作流程

```python
# 1. 创建页面对象
page = ChromiumPage()

# 2. 访问网页
page.get('https://www.example.com')

# 3. 查找元素
ele = page.ele('#selector')

# 4. 元素操作
ele.click()
ele.input('text')
```
