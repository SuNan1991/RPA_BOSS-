# 浏览器启动设置

来源: https://drissionpage.cn/dp40docs/ChromiumPage/browser_opt

浏览器的启动配置非常繁杂，本库使用 `ChromiumOptions` 类管理启动配置，并且内置了常用配置的设置接口。

**注意**:
- 该对象只能用于浏览器的启动
- 浏览器启动后，再修改该配置没有任何效果
- 接管已打开的浏览器时，启动配置也是无效的

---

## 创建对象

### 导入

```python
from DrissionPage import ChromiumOptions
```

### 初始化参数

`ChromiumOptions` 对象用于管理浏览器初始化配置，可从配置文件中读取配置来进行初始化。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `read_file` | `bool` | `True` | 是否从 ini 文件中读取配置信息 |
| `ini_path` | `Path` `str` | `None` | 指定 ini 文件路径 |

**创建配置对象**:

```python
from DrissionPage import ChromiumOptions

co = ChromiumOptions()
```

默认情况下，`ChromiumOptions` 对象会从 ini 文件中读取配置信息，当指定 `read_file` 参数为 `False` 时，则以默认配置创建。

---

## 使用方法

创建配置对象后，可调整配置内容，然后在页面对象创建时以参数形式把配置对象传递进去。配置对象支持链式操作。

```python
from DrissionPage import WebPage, ChromiumOptions

# 创建配置对象（默认从 ini 文件中读取配置）
co = ChromiumOptions()
# 设置不加载图片、静音
co.no_imgs(True).mute(True)
# 以该配置创建页面对象
page = WebPage(chromium_options=co)
```

```python
from DrissionPage import ChromiumOptions, ChromiumPage

co = ChromiumOptions()
co.incognito()  # 匿名模式
co.headless()  # 无头模式
co.set_argument('--no-sandbox')  # 无沙盒模式
page = ChromiumPage(co)
```

## 命令行参数设置

Chromium 内核浏览器有一系列的启动配置，以 `--` 开头，可在浏览器创建时传入，控制浏览器行为和初始状态。

### set_argument()

此方法用于设置启动参数。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `arg` | `str` | 必填 | 启动参数名称 |
| `value` | `str` `None` `False` | `None` | 参数的值 |

**示例**:

```python
# 设置启动时最大化
co.set_argument('--start-maximized')
# 设置初始窗口大小
co.set_argument('--window-size', '800,600')
# 使用来宾模式打开浏览器
co.set_argument('--guest')
```

### remove_argument()

此方法用于删除一个启动参数。

```python
# 删除--start-maximized参数
co.remove_argument('--start-maximized')
```

### clear_arguments()

此方法用于清空已设置的 `arguments` 参数。

## 运行路径及端口

### set_browser_path()

设置浏览器可执行文件路径。

```python
co.set_browser_path(r'D:\chrome.exe')
```

### set_tmp_path()

设置临时文件存放路径。

### set_local_port()

设置本地启动端口。

```python
co.set_local_port(9222)
```

### set_address()

设置浏览器地址，格式 'ip:port'。

```python
co.set_address('127.0.0.1:9222')
```

### auto_port()

设置是否使用自动分配的端口，启动一个全新的浏览器。

```python
co.auto_port(True)
```

**注意**: `auto_port()` 支持多线程，但不支持多进程。多进程使用时，可用 `scope` 参数指定每个进程使用的端口范围。

### set_user_data_path()

设置用户文件夹路径。

### use_system_user_path()

设置是否使用系统安装的浏览器默认用户文件夹。

### set_cache_path()

设置缓存路径。

### existing_only()

设置是否仅使用已启动的浏览器。

## 使用插件

### add_extension()

添加一个插件到浏览器。

```python
co.add_extension(r'D:\SwitchyOmega')
```

### remove_extensions()

移除配置对象中保存的所有插件路径。

```python
co.remove_extensions()
```

## 用户文件设置

### set_user()

选择使用哪一个用户配置，默认为 'Default'。

```python
co.set_user(user='Profile 1')
```

### set_pref()

设置用户配置文件里的一个配置项。

```python
# 禁止所有弹出窗口
co.set_pref(arg='profile.default_content_settings.popups', value='0')
# 隐藏是否保存密码的提示
co.set_pref('credentials_enable_service', False)
```

### remove_pref()

删除当前配置对象中的一个 `pref` 配置项。

### remove_pref_from_file()

在用户配置文件删除一个配置项。

### clear_prefs()

清空已设置的 `prefs` 参数。

---

## 运行参数设置

### set_timeouts()

设置几种超时时间，以秒为单位。

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `base` | `float` | `None` | 默认超时时间 |
| `page_load` | `float` | `None` | 页面加载超时时间 |
| `script` | `float` | `None` | JavaScript 运行超时时间 |

```python
co.set_timeouts(base=10)
```

### set_retry()

设置页面连接超时时的重试次数和间隔。

### set_load_mode()

设置网页加载策略。

加载策略：
- `'normal'`：阻塞进程，等待所有资源下载完成（默认）
- `'eager'`：DOM 就绪即停止加载
- `'none'`：网页连接成功即停止加载

```python
co.set_load_mode('eager')
```

### set_proxy()

设置浏览器代理。

```python
co.set_proxy('http://localhost:1080')
```

### set_download_path()

设置下载文件保存路径。

## 其它设置

### headless()

设置是否以无界面模式启动浏览器。

```python
co.headless(True)
```

### set_flag()

设置实验项，即 `'chrome://flags'` 中的项目。

```python
co.set_flag('temporary-unexpire-flags-m118', '1')  # 有值
co.set_flag('disable-accelerated-2d-canvas')  # 无值
```

### clear_flags_in_file()

删除浏览器配置文件中已设置的实验项。

### clear_flags()

清空本对象中已设置的 `flags` 参数。

### incognito()

设置是否以无痕模式启动浏览器。

```python
co.incognito(True)
```

### ignore_certificate_errors()

设置是否忽略证书错误。

### no_imgs()

设置是否禁止加载图片。

```python
co.no_imgs(True)
```

### no_js()

设置是否禁用 JavaScript。

### mute()

设置是否静音。

### set_user_agent()

设置 user agent。

```python
co.set_user_agent(user_agent='Mozilla/5.0...')
```

### set_paths()

设置各种路径信息。

```python
co.set_paths(local_port=9333, user_data_path=r'D:\tmp')
```

## 保存设置到文件

### save()

保存配置项到一个 ini 文件。

```python
# 保存当前读取的ini文件
co.save()
# 把当前配置保存到指定的路径
co.save(path=r'D:\tmp\settings.ini')
```

### save_to_default()

保存配置项到固定的默认 ini 文件。

## ChromiumOptions 属性

- `address` - 浏览器地址
- `browser_path` - 浏览器可执行文件路径
- `user_data_path` - 用户数据文件夹路径
- `tmp_path` - 临时文件夹路径
- `download_path` - 默认下载路径
- `user` - 用户配置文件夹名称
- `load_mode` - 页面加载策略
- `timeouts` - 超时设置
- `retry_times` - 连接失败重试次数
- `retry_interval` - 连接失败重试间隔
- `proxy` - 代理设置
- `arguments` - 浏览器启动参数列表
- `extensions` - 要加载的插件路径列表
- `preferences` - 用户首选项配置
- `system_user_path` - 是否使用系统浏览器用户文件夹
- `is_existing_only` - 是否仅使用已打开的浏览器
- `is_auto_port` - 是否自动分配端口和用户文件夹路径
